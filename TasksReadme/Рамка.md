# Работа по написаю программы для вывода окна с текстом в виртуальной среде `DosBOX`

# 1. Цель работы

Разработка программы для вывода окна с текстом. Реализация задания параметров окна через аргументы командной строки.

# 2. Функционал программы

Разработанная программа поддерживает следующие аргументы командной строки:
- `-x=`, `-y=` - координаты верхнего левого угла окна.
- `-f=`, `-r=` - настройка цвета фона и окрамления окна соотвественно.
- `text + $` - задание одной строчки выводимого текста. `text` - в этом случае не кодовое слово, а выводимый текст. За окочание строки отвечает символ `$`. Строки будут выводится в обратном порядке с выравниванием посередине.

После каждого аргумента сразу за ним должно идти 16-тиричное число, отвечающее за значение аргумента при работе программы.

```
Важное дополнение! Числом считаются два символа, идущие сразу после `=`. Поэтому, если поставить пробел или написать только один символ, вы можете получить не тот результат, который ожидали.
```

# 3. Запуск и использование

Программа представляет собой файл `102.com`, хранящийся в папке `src/Rep/`.
Для её запуска необходимо набрать соотвествующий адрес в командной строке.


Эта программа предназначена для отображения рамки с информацией в консоли Dosbox.

# 4. Примеры вызова

`102.com 45678$ -y=01 -x=11 -r=03 -f=06`
